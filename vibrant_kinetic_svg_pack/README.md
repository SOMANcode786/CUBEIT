# Vibrant Kinetic SVG Pack

18 original, fully editable SVG glyphs inspired by the vibrant floating geometric/doodle language in the reference animation.

- Transparent backgrounds
- 128×128 viewBox
- Vector paths/shapes only
- Gradient-based vibrant palette
- Ideal for React/Next.js kinetic particle fields
- Safe to recolor, animate, rotate, scale, or use as masks

Usage:
<img src="/particles/01_folded_bow.svg" alt="" />

For inline control, paste the SVG markup into a React component and animate the wrapper using transform: translate3d(...) rotate(...).

These assets intentionally match the reference's visual language without being pixel-for-pixel copies of another designer's artwork.
